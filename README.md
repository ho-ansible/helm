# Ansible role: helm
[Helm](https://helm.sh) manages software releases installed on a kubernetes cluster.

## Requirements
Only tested on Debian stable, for now.

## Role Variables

## Playbooks
+ `main.yml`: apply role
+ `uninstall.yml`: remove. Run before removing config from inventory.
